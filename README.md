# 📖 Project Overview

The **Cyber Security Incident Analysis Dashboard** is an interactive Tableau dashboard designed to transform raw cybersecurity incident data into meaningful business insights. The dashboard provides a comprehensive view of cyber attack trends, financial impact, affected users, attack severity, and industry-wise analysis, enabling organizations to make informed decisions and strengthen cybersecurity strategies.

---

# ❗ Problem Statement

Cyber security incidents are increasing across industries, leading to substantial financial losses, compromised sensitive data, and operational disruptions. While organizations collect large volumes of incident data, extracting actionable insights from raw datasets is often challenging.

Without effective visualization, it becomes difficult to:

- Monitor attack trends over time.
- Identify industries at higher risk.
- Measure the financial impact of cyber attacks.
- Understand attack severity and distribution.
- Track affected users and compromised data.
- Analyze geographical patterns of cyber incidents.

An interactive dashboard is required to consolidate these insights into a single, easy-to-understand platform for business users and decision-makers.

---

# 💡 Solution

Developed an interactive Tableau dashboard that converts raw cybersecurity incident data into meaningful visual insights using KPIs and analytical charts.

The dashboard enables stakeholders to:

- Monitor cyber security incidents in real time.
- Analyze financial losses caused by cyber attacks.
- Identify high-risk industries.
- Track monthly attack trends.
- Understand attack type distribution.
- Evaluate severity levels.
- Visualize geographical distribution of incidents.
- Measure affected users and compromised records.

# 📊 Key Performance Indicators (KPIs)

| KPI | Description |
|------|-------------|
| Total Incidents | Total number of reported cyber security incidents |
| Total Financial Loss | Overall financial loss caused by cyber attacks |
| Critical Attack Count | Number of incidents classified as Critical |
| Data Compromised | Total compromised records |
| Average Financial Loss | Average financial loss per incident |
| Affected Users | Total users impacted by cyber attacks |

# 📌 Key Insights

- A total of **1,500** cyber security incidents were analyzed.
- The total financial loss exceeded **$493.5 Million**.
- More than **732 Million** records were compromised.
- Over **37.6 Million** users were affected.
- The **Banking** industry recorded the highest financial losses.
- High and Medium severity incidents accounted for a significant portion of cyber attacks.
- Cyber attacks were distributed across multiple countries, highlighting the global impact of cybersecurity threats.
